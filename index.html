1. What is Full Stack Development?

Full Stack Development means building both the front end, which is what users see and back end, which is what happens behind the sences like server and database of websites.

For example a restaurant:

  • Front End: Area where customers sit and eat (HTML, CSS, JavaScript)
  • Back End: Kitchen where food is prepared (servers, databases)
  • Full Stack Developer: Someone who can design and manage servers, databases

2. HTML (HyperText Markup Language)

HTML (Hyper Text Markup Language) is the skeleton of web pages. It uses tags to shape the structure of the content. It’s not a programming language and it doesn’t handle logic or make things happen but it gives your content shape like texts, headings, images, links, and more.

Basic HTML Structure that I Used:
<!DOCTYPE html>
<html lang="en">
<head>
    <title>My Portfolio</title>
    <meta charset="UTF-8">
</head>
<body>
    <h1>My Heading</h1>
    <p>My paragraph text</p>
</body>
</html>
HTML Tags Explained:
  • <html> - Contains everything
  • <head> - Information about the page
  • <body> - The visible content
  • <h1> - Main heading (Biggest)
  • <p> - Paragraph text
  • <div> - Container for grouping elements

3. CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is a style sheet language which makes websites look good by controlling colors, fonts, layout, and animations. Basically, CSS designs the appearance of a site after HTML defines its structure.

CSS that I Used for This Page:
body {
    font-family: 'Arial', sans-serif;
    background: #1a1a1a;
}

.header {
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
    color: white;
    padding: 40px;
    text-align: center;
    border-radius: 15px;
}
CSS Properties Explained:
  • background: - Sets background color or image
  • color: - Changes text color
  • padding: - Space inside an element
  • margin: - Space outside an element
  • border-radius: - Makes rounded corners
  • font-family: - Changes the font

4. Bootstrap Framework

Bootstrap is a CSS framework that provides pre made styles and components to build websites faster. It gives you pre made HTML, CSS, and JavaScript pieces like buttons, forms, grids, etc.

Bootstrap classes that I have used:
<div class="container">
    <div class="row">
        <div class="col-md-6">Left Column</div>
        <div class="col-md-6">Right Column</div>
    </div>
</div>

<button class="btn btn-primary">Bootstrap Button</button>
Bootstrap Classes Explained:
  • .container - Centers content with proper spacing
  • .row - Creates a horizontal row
  • .col-md-6 - Takes up 6/12 columns (half width) on medium+ screens
  • .btn - Makes a nice button
  • .btn-primary - Blue button style

5. JavaScript - Making Websites Interactive

JavaScript makes websites interactive, it is a programming language that brings web pages to life by making them interactive and dynamic. It can respond to user actions and change content dynamically.

JavaScript Functions that I have wrote:
function changeBackgroundColor() {
    const colors = ['#1a1a1a', '#2c2c2c', '#3d3d3d'];
    const randomColor = colors[Math.floor(Math.random() * colors.length)];
    document.body.style.backgroundColor = randomColor;
}

function showAlert() {
    alert('JavaScript in action!');
}
JavaScript Concepts Explained:
  • function - A block of code that does something
  • document - Represents the webpage
  • getElementById() - Finds an element by its ID
  • addEventListener() - Waits for user actions (clicks, etc.)
  • Math.random() - Generates random numbers
